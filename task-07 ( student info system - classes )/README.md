# 🎓 C++ Student Class System

![Language](https://img.shields.io/badge/Language-C%2B%2B-blue.svg)
![Level](https://img.shields.io/badge/Level-Beginner-success)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

A beginner-friendly C++ project that introduces the fundamentals of **Object-Oriented Programming (OOP)** by creating a simple `student` class. The program demonstrates how to store and manage student information using **private data members**, **getter functions**, and **setter functions**.

This project was built as a practice exercise to understand encapsulation and class design before moving on to constructors, inheritance, and more advanced OOP concepts.

---

## ✨ Features

* Create a custom `student` class
* Store student information using private data members
* Modify student information using setter functions
* Access stored information using getter functions
* Display all student information using a member function
* Practice encapsulation
* Demonstrate object creation and usage

---

## 🖥️ Program Flow

The program performs the following operations:

```text
1. Create a student object
2. Set the student's:
      • Name
      • Age
      • Student ID
      • Average
      • Letter Grade
3. Store the data inside the object
4. Display all student information
```

## 📦 Member Functions Used

| Function         | Description                           |
| ---------------- | ------------------------------------- |
| `setName()`      | Sets the student's name               |
| `setAge()`       | Sets the student's age                |
| `setstudentId()` | Sets the student ID                   |
| `setAverage()`   | Sets the student's average            |
| `setGrade()`     | Sets the student's letter grade       |
| `getName()`      | Returns the student's name            |
| `getAge()`       | Returns the student's age             |
| `getID()`        | Returns the student ID                |
| `getAvg()`       | Returns the student's average         |
| `getGrade()`     | Returns the student's letter grade    |
| `printInfo()`    | Prints all stored student information |
