## 📖 Overview

This project is my **first assignment** in the **ITI Embedded Systems Track**.

The program is a simple command-line calculator written in **C++** that performs basic arithmetic operations while demonstrating fundamental programming concepts such as loops, conditional statements, input validation, and `switch` statements.

---

## ✨ Features

✅ Addition (`+`)

✅ Subtraction (`-`)

✅ Multiplication (`*`)

✅ Division (`/`)

✅ Modulus (`%`)

✅ Operator validation

✅ Integer-only modulus operation

✅ Division-by-zero protection

## 📷 Example

### ➕ Addition

```text
Enter an operator (+, -, *, /, %): +

+ operator can work with either integers or floats

Enter first number:
10.5

Enter second number:
2

Result: 12.5
```

### ➗ Division

```text
Enter an operator (+, -, *, /, %): /

Enter first number:
25

Enter second number:
5

Result: 5
```

### ➖ Modulus

```text
Enter an operator (+, -, *, /, %): %

Modulus operator works only with integers.

Enter first number:
10

Enter second number:
3

Result: 1
```

## ⚠️ Notes

- `%` works **only with integers** in C++.
- The program keeps asking until a **valid operator** is entered.
- Division by zero is handled safely.
